Troubleshooting & FAQ

Support • 8 min read

This guide covers common issues and frequently asked questions about Studo. If you don't find your issue here, please contact support for personalized assistance.

Timer Issues

Timer won't start or stops unexpectedly

Possible causes and solutions:

  • Browser tab inactive: Some browsers pause JavaScript when tabs are inactive. Keep the tab active or use a different browser.
  • Browser permissions: Ensure your browser allows JavaScript and local storage.
  • Cache issues: Clear your browser cache and reload the page.
  • Multiple tabs: Avoid running multiple timer instances in different tabs.
Timer shows incorrect time or resets

Solutions:

  • Check your device's time and date settings
  • Clear browser cache and reload the page
  • Ensure you're not running multiple instances
  • Try using a different browser
Pomodoro timer not working (Pro version)

Troubleshooting steps:

  • Verify you're logged into your Pro account
  • Check your internet connection for cloud sync
  • Clear browser cache and reload
  • Ensure browser notifications are enabled

Data and Storage Issues

My study data disappeared

For Free Version:

  • Check if you cleared browser data recently
  • Verify you're using the same browser and device
  • Check if you're in incognito/private mode
  • Look for data in browser's local storage settings

For Pro Version:

  • Ensure you're logged into the correct account
  • Check your internet connection
  • Try logging out and back in
  • Contact support for data recovery assistance
Heatmap not showing or updating

Solutions:

  • Refresh the page to reload the heatmap
  • Check if you have any logged study sessions
  • Clear browser cache and reload
  • Ensure JavaScript is enabled
  • Try a different browser
Can't add notes or images to sessions

Troubleshooting:

  • Ensure you've completed a study session first
  • Check if you're in the session logging interface
  • Verify browser permissions for file uploads
  • Try refreshing the page
  • Check available storage space (Pro version)

Account and Subscription Issues

Can't log into Pro account

Solutions:

  • Verify your email and password
  • Check your internet connection
  • Try resetting your password
  • Clear browser cache and cookies
  • Disable browser extensions temporarily
  • Try a different browser or device
Pro features not working after payment

Steps to resolve:

  • Wait 5-10 minutes for payment processing
  • Log out and log back into your account
  • Check your email for payment confirmation
  • Verify your subscription status in account settings
  • Contact support with your payment receipt
Data not syncing across devices (Pro)

Troubleshooting:

  • Ensure you're logged into the same account on all devices
  • Check internet connection on all devices
  • Force refresh the app on each device
  • Check if sync is enabled in settings
  • Try logging out and back in on all devices

Performance and Browser Issues

App is slow or unresponsive

Performance optimization:

  • Close unnecessary browser tabs
  • Clear browser cache and cookies
  • Disable browser extensions temporarily
  • Restart your browser
  • Check available RAM and CPU usage
  • Try using a different browser
App doesn't work on mobile devices

Mobile compatibility:

  • Use a modern mobile browser (Chrome, Safari, Firefox)
  • Ensure JavaScript is enabled
  • Check if you're in private/incognito mode
  • Clear browser cache and data
  • Update your mobile browser to the latest version
  • Try adding the app to your home screen

Flashcard Issues (Pro)

Flashcards not saving or syncing

Solutions:

  • Ensure you're logged into your Pro account
  • Check your internet connection
  • Verify you have sufficient storage space
  • Try creating a new flashcard set
  • Clear browser cache and reload
Spaced repetition not working correctly

Troubleshooting:

  • Ensure you're rating cards consistently
  • Check if you have enough cards in your deck
  • Verify your review settings
  • Try resetting the spaced repetition algorithm
  • Contact support for algorithm-specific issues

General FAQ

Is my data safe and private?

Yes, your data is completely private:

  • Free version: All data stays on your device
  • Pro version: Data is encrypted and stored securely
  • We never share or sell your data
  • You can export or delete your data anytime
Can I use Studo offline?

Offline functionality:

  • Free version: Works completely offline
  • Pro version: Basic features work offline, sync when online
  • Timer and session logging work without internet
  • Cloud features require internet connection
How do I export my data?

Export options:

  • Free version: Use browser's export local storage feature
  • Pro version: Export from account settings in CSV/PDF format
  • All session data, notes, and analytics can be exported
  • Contact support for bulk export assistance
What browsers are supported?

Supported browsers:

  • Chrome (recommended)
  • Firefox
  • Safari
  • Edge
  • Opera

Use the latest version for best performance.

Getting Help

Still need help?

If you can't find a solution here:

  • Contact Support: Reach out through the app or website
  • Include Details: Describe the issue, browser, and steps to reproduce
  • Response Time: We typically respond within 24 hours
  • Pro Users: Priority support for Pro subscribers

Before Contacting Support

  • Try the solutions in this guide
  • Clear your browser cache
  • Try a different browser
  • Check your internet connection
  • Note your browser version and operating system
